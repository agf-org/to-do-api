# To-do API

## Usage

Create a `.env` file inside the project directory with the following environment variables:
```
NODE_ENV=local
PORT=3000
TO_DO_API_TAG=local
DEBUG=to-do-api:*
```

To start the app use:
```
$ npm start
```

## Documentation

API specification can be found at http://127.0.0.1:3000/api/api-docs/
